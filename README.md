# TechMart E-Commerce Store

## Overview

TechMart is a simple E-Commerce Store developed using HTML, CSS, JavaScript, Node.js, and Express.js. The project provides a user-friendly interface for browsing products, adding items to a shopping cart, viewing the total amount, and placing orders.

## Features

* Product Listing Page
* Shopping Cart Functionality
* Add to Cart Feature
* Total Price Calculation
* Order Processing
* Responsive User Interface
* Modern E-Commerce Design

## Technologies Used

* HTML5
* CSS3
* JavaScript
* Node.js
* Express.js

## Installation

1. Clone the repository

```bash
git clone https://github.com/your-username/ecommerce-store.git
```

2. Open the project folder

```bash
cd ecommerce-store
```

3. Install dependencies

```bash
npm install
```

4. Start the server

```bash
node server.js
```

5. Open the browser

```text
http://localhost:3000
```
